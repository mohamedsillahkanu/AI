ℹ️ How It Works

Column Renaming:

  1. Upload File: CSV or Excel file with your data
  2. View Columns: See all column names with their data types
  3. Enter New Names: Type new names for columns you want to rename
  4. Apply Changes: Generate complete dataset with new column names

Value Renaming:

  1. Select Column: Choose which column's values to rename
  2. View Unique Values: See all unique values and their frequencies
  3. Rename Values: Enter new names for specific values
  4. Bulk Operations: Use find/replace and transformations
